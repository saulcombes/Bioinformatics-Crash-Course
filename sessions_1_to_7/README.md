# Sessions 1–7: FASTQ → QC → Trimming → Synthetic Reads

This folder contains all code, notes, and deliverables from Sessions 1–7 of my bioinformatics crash course. These sessions were completed as part of a single integrated project focused on understanding and processing FASTQ sequencing data from first principles.
Across these sessions, I built a Python-based pipeline that covers:

Core Topics Covered
- FASTQ structure and parsing
- Phred quality scores and quality filtering
- Basic QC metrics and read statistics
- Synthetic read generation for testing
- Adapter contamination detection
- Trimming logic (synchronised sequence + quality trimming)
- Documentation of limitations and future improvements

## What This Folder Represents
Rather than separating each session into its own directory, Sessions 1–7 are grouped here because they formed one continuous learning project. The work evolved naturally from simple parsing to a functional trimming/QC pipeline, so this folder reflects that progression.
